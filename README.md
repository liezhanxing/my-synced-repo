# 小朋友的英文字母学习游戏

## 游戏简介
这是一个专为小朋友设计的英文字母学习游戏，具有活泼可爱的界面和趣味性的交互元素。

## 游戏特色
- 生动活泼的界面设计，色彩丰富
- 有趣的动画效果
- 互动式字母识别练习
- 即时反馈和鼓励机制
- 循序渐进的学习模式

## 游戏玩法
1. **字母展示阶段**：屏幕上会显示一个大大的英文字母
2. **测验阶段**：点击按钮进入测验，找出提示的字母
3. **结果反馈**：显示正确或错误的反馈，并奖励分数
4. **循环学习**：自动进入下一个字母的学习

## 运行方式

### 方式一：直接在浏览器中打开（推荐）
1. 下载整个项目文件夹
2. 双击 `index.html` 文件
3. 游戏将在浏览器中自动打开

### 方式二：通过本地服务器运行
1. 在项目目录下打开终端
2. 运行命令：`python -m http.server 8000`
3. 在浏览器中访问：http://localhost:8000

## 技术栈
- HTML5
- CSS3
- JavaScript (ES6)
