# 小朋友的英文字母学习游戏

## 游戏简介
这是一个专为小朋友设计的英文字母学习游戏，具有活泼可爱的界面和趣味性的交互元素。

## 游戏特色
- 生动活泼的界面设计，色彩丰富
- 有趣的动画效果（漂浮气泡、闪烁星星）
- 互动式字母识别练习
- 即时反馈和鼓励机制
- 循序渐进的学习模式

## 游戏玩法
1. **字母展示阶段**：屏幕上会显示一个大大的英文字母
2. **测验阶段**：点击空格键进入测验，找出提示的字母
3. **结果反馈**：显示正确或错误的反馈，并奖励分数
4. **循环学习**：自动进入下一个字母的学习

## 操作说明
- **空格键**：开始测验或继续下一个字母
- **鼠标点击**：在测验阶段选择正确的字母

## 运行要求
- Python 3.x
- Pygame库

## 安装运行
1. 确保已安装Python 3.x
2. 安装pygame库：`pip install pygame`
3. 运行游戏：`python alphabet_game.py`
